# 🇬🇧 English

# TURBO RACE 3D

A modern 3D-style racing game built with Python and PyQt6 featuring smooth gameplay, multilingual support, dynamic themes, AI opponents, nitro boost mechanics, animated HUD elements, and responsive controls.

## Features

- 3D-inspired racing experience
- Smooth real-time rendering using PyQt6
- AI-controlled opponents
- Nitro boost system
- Lap tracking and scoring system
- Best lap timer
- Dynamic speedometer
- Dark and Light themes
- Multilingual interface:
  - English
  - Persian (فارسی)
  - Chinese (中文)
- Responsive keyboard controls
- Modern HUD and visual effects
- Animated particles and lighting effects

## Requirements

- Python 3.10 or newer
- PyQt6

## Installation

### Install Required Libraries

```bash
pip install PyQt6
```

### Alternative

```bash
pip3 install PyQt6
```

## Run

```bash
python "TURBO RACE 3D.py"
```

## Controls

| Key | Action |
|------|---------|
| ↑ / W | Accelerate |
| ↓ / S | Brake |
| ← / A | Move Left |
| → / D | Move Right |
| Space | Nitro Boost |

## Game Systems

### Racing
Complete laps before your opponents and achieve the highest score.

### Nitro
Use Space to activate Nitro for a temporary speed boost.

### Scoring
Points are awarded based on:

- Race performance
- Lap completion
- Fast lap times
- Final race results

### Themes
Switch between Dark and Light themes at any time.

### Languages
Change the game language instantly from the settings controls.

# TURBO RACE 3D

一款使用 Python 与 PyQt6 开发的现代化 3D 风格赛车游戏，拥有流畅动画、多语言支持、AI 对手、氮气加速系统以及精美界面。

## 功能特点

- 3D 风格赛车体验
- PyQt6 实时渲染
- AI 对手系统
- 氮气加速功能
- 圈数统计
- 最佳圈速记录
- 动态速度表
- 深色与浅色主题
- 多语言支持：
  - English
  - فارسی
  - 中文
- 键盘控制
- 现代化 HUD
- 粒子与光效动画

## 环境要求

- Python 3.10 或更高版本
- PyQt6

## 安装

### 安装所需库

```bash
pip install PyQt6
```

或

```bash
pip3 install PyQt6
```

## 运行游戏

```bash
python "TURBO RACE 3D.py"
```

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 加速 |
| S / ↓ | 刹车 |
| A / ← | 左转 |
| D / → | 右转 |
| Space | 氮气加速 |

## 游戏系统

### 比赛模式

完成比赛圈数并获得更高分数。

### 氮气系统

按下 Space 启动氮气加速。

### 计分系统

分数来源：

- 比赛表现
- 完成圈数
- 最佳圈速
- 最终排名

### 主题

支持深色和浅色主题切换。

### 语言

游戏内可实时切换语言。
